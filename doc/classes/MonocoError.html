    Defined in src/workflow.js:439

    Change the state of a component.

    Worklow:

    1. Check if the component has not been destroyed
    2. Check if the state is a method or an event
    3. Search if there is a behavior with an action for the new state
    4. If so, get the action(s)
    5. Check if the conditons on input are compliant with the metamodel
    6. Call the action(s)
    7. If not an of event, check if the conditons on input are compliant with the metamodel
    8. If all is ok, the state of the component is updated
    9. Return the result

    Parameters:

    • params Object

      params to change the state
      {String} component id of the component
      {String} state state of the component
      {Array} data parameters to send to the action
