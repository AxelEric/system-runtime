Example 08 - Package a System

We use the system of example 04.
We saved the exported system in a JSON file and put it on the "addons" directory of monoco (/src/addons/).
We change the "component" property of the "main" action of the system to the one of monoco, "e89c617b6b15d24" (this manual step will be removed on a future version of monoco).
Then when building monoco, the system will be imported in monoco system and monoco will execute the main function of the imported system.

Look at the console of your browser to see the result of packaged system.
