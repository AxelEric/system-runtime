Example 07 - Load the system from HTML

We use the system of example 04.
We saved the exported system in a JSON file and then we loaded it with a link tag.
The main method of the system will be called when the system will be imported.

Look at the console of your browser to see the result of the imported system.
NB: an error can occured if you access this file from the file system. 

<link rel="system" type="application/json" href="system/example04.json">
